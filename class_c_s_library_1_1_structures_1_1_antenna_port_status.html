 The stored value from the last measurement of the antenna-sense resistor for the logical antenna port's physical transmit antenna port. The last measurement taken occurred the last time that the carrier wave was turned on for this antenna port – note that this means that when retrieving the logical antenna port's status, this does not Result in an active measurement of the antenna-sense resistor. This value is specified in ohms.
