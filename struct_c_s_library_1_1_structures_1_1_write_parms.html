Detailed Description

A tag-write command allows an application to write one or more 16-bit words to the specified memory bank of the ISO 18000-6C tags of interest. The data returned from the tag write will adhere to the format specified in [REC]. An application is given some flexibility in the execution of a write to an ISO 18000- 6C tag.
