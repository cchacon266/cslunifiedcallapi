Detailed Description

A tag block-write command allows an application to write one or more 16-bit words to the specified memory bank of the ISO 18000-6C tags of interest. The data returned from the block write will adhere to the format specified in [REC]. A Write can be performed on a contiguous set of one or more 16-bit words to one of the tag's memory banks.
