 A buffer of count 16-bit values to be written sequentially to the tag's specified memory bank. The high-order byte of pData[n] will be written to the tag's memory-bank byte at 16-bit offset (offset+n). The low-order byte will be written to the next byte. For example, if offset is 2 and pData[0] is 0x1122, then the tag-memory byte at 16-bit offset 2 (byte offset 4) will have 0x11 written to it and the next byte (byte offset 5) will have 0x22 written to it.
